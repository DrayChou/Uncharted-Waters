# 二代 人物 + 港口 + 船只资料

> 同名 PDF（[`人物港口船只资料.pdf`](人物港口船只资料.pdf)，4 页 Letter，约 3.4 MB）是 macOS Numbers 导出的大尺寸数据表，建议直接打开 PDF 查看完整数据。这里只做内容索引。

## PDF 内容结构

PDF 分 4 页，分别承载 4 个表格：

### 第 1-2 页 · 人物角色 / NPC 一览

每个角色一行，列：

| 字段 | 说明 |
|---|---|
| 角色头像 | 游戏内立绘 / 头像 |
| 中文名 | 中文翻译名 |
| 英文 / 原名 | 游戏文本中的原始名称 |
| 国籍 | 葡萄牙、西班牙、英国、土耳其、日本、明朝… |
| 身份 | 主角、酒馆 NPC、提督、商人、海盗… |
| 所在港口 | 主要活动港口（如里斯本、伊斯坦布尔、长崎、长安等） |
| 备注 | 剧情触发、加入条件、特长等 |

主要主角：
- **若阿娜**（Joao Felinho）- 葡萄牙
- **卡塔利娜**（Catalina Erantzo）- 西班牙
- **皮埃尔**（Pietro Conti）- 意大利商人
- **欧塞·特·拉佩鲁兹**（Aussel）- 法国
- **阿尔·瓦伊塔**（Ali Vaiteh）- 土耳其
- **奥东·萨摩**（Otto Spinola）- 神圣罗马
- F5+ 加强版新增：**张洪图（大明）**、**米兰达（外传）**

### 第 3 页 · 港口 / 街道一览

每个港口一行，列：

| 字段 | 说明 |
|---|---|
| 港口名 | 中文 + 拉丁字母 |
| 国籍 / 势力 | 所属阵营 |
| 经度 / 纬度 | 地图坐标 |
| 特产 / 名产 | 该港主要商品 |
| 进货价 / 出货价 | 基准价位 |
| 街道布局 | 同级目录 [`../港口布局图/`](../港口布局图/) 有 100 张高清布局 |
| 建筑物 | 银行、酒馆、船厂、武器店、道具店等 |
| 备注 | 特殊事件、捕鱼场、宝藏起点等 |

港口大致按地区分组：

- **欧洲**：里斯本、塞维利亚、伦敦、阿姆斯特丹、伊斯坦布尔…
- **地中海**：威尼斯、雅典、亚历山大、阿尔及尔…
- **北非 / 西非**：突尼斯、马德拉、佛得角、莫桑比克…
- **印度洋**：阿丁、霍尔木兹、卡利卡特、果阿…
- **东亚**：长安、长崎、堺、马尼拉、台南…
- **新大陆**：圣多明各、卡塔赫纳、利马、布宜诺斯艾利斯…

### 第 4 页 · 船只规格

每种船一行，列：

| 字段 | 说明 |
|---|---|
| 船图 | sprite 缩略图（同级目录 [`../船只图片/`](../船只图片/) 有 25 种） |
| 船名 | 中文 + 原文 |
| 转向 / 推进 | 海战机动性，见 [`海战中的转向与推进.md`](海战中的转向与推进.md) |
| 耐久 | 抗损值 |
| 水手数 | 最少 / 最大 |
| 武器 | 大炮数量上限 |
| 容积 | 货物 + 食粮 + 水的总容量 |
| 帆型 | 横帆 / 纵帆 / 混合 / 桨 |
| 抗暴 | 抗风暴能力 |
| 造价 | 船厂购入价 |
| 天数 | 订造交付天数 |
| 工业值 | 所在港口工业值需求 |
| 港口 | 可买到的港口 |
| 介绍 | 简短背景说明 |

原版有 25 种基础船型；F5+ 加强版扩展到 37 种，新规格见 [`../强化版/F5+系统详解.md`](../强化版/F5+系统详解.md)。

## 阅读建议

1. **角色 / 港口查询**：直接打开 PDF，用 macOS 预览的搜索（`Cmd-F`）即可全文检索中文 / 原文。
2. **船只比较**：除了 PDF 的表格，建议同时打开 [`../船只图片/README.md`](../船只图片/README.md) 看 sprite 图。
3. **看不清 PDF**：PDF 是矢量图，缩放无损；如果浏览器渲染模糊，下载到本地用 macOS 预览或 PDF 阅读器打开。
