## 简介

原帖是 2007 年飘雪模拟社区 KT-7EX 整理的**二代 25 种基础船型完整数据表**，2018 年由 compass 在意志之路论坛转帖。**内容为图片（影印版）**，PDF 中的文字无法直接复制，需要用看图方式查阅。

## 数据表内容

每个船型一行，包含以下字段：

| 字段 | 说明 |
|---|---|
| 船名 | 中文 + 原文 |
| 转向 / 推进 | 海战机动性 |
| 耐久 | 抗损值 |
| 武器 / 最大武器 | 大炮数量与上限 |
| 水手 / 最大水手 | 标配与上限 |
| 容积 | 货舱 + 食水容量 |
| 帆型 | 横帆 / 纵帆 / 混合 |
| 抗暴 | 抗风暴能力 |
| 造价 | 船厂购入价 |
| 订造天数 | 下单到交货 |
| 工业值要求 | 该港需要的最低工业值 |
| 可买港口 | 哪些港口有售 |
| 介绍 | 简短背景 |

> **红色数据** = 意志加强版修正后值
> **意志新增船型** = 1.9E 之后新增的型号（原版没有）
