# 二代 逆向资料

这里整理的是从 [`dongzhang84/sea2_demo`](https://github.com/dongzhang84/sea2_demo) 提炼出来、并且**适合本仓库收纳**的《大航海时代 II》补充资料。

## 这部分是什么

`sea2_demo` 本体是一个以 **PC-9801《大航海时代 II》资源逆向** 为核心的项目，内容很丰富，包含：

- 资源解包与渲染脚本
- 港口 / 世界地图 / 船只等导出数据
- 剧情与事件结构分析文档
- Godot 4 的 demo 原型

但本仓库的定位是 **资料库**，不是逆向工程代码仓，也不是复刻游戏工程。因此这里只吸收最有检索价值、最适合玩家和整理者直接使用的部分。

## 为什么没有整包搬运

以下内容虽然在 `sea2_demo` 里很重要，但**不适合直接并入**本仓库：

1. **Godot demo 工程代码本身**
   - 这是独立项目，不属于本仓库“资料整理”主线；但其中对复刻有直接价值的图片素材已单独导入到 `assets/sea2_demo/`。
2. **用于 demo 的派生平衡数据**
   - 例如 `game/data/ports.json` 里的商品价格、开局资金、造船厂配置，更偏 demo 玩法，不宜当作原版权威资料。
3. **尚未完全解码的半成品数据**
   - 例如部分船名、角色名仍保留十六进制或占位文本，直接收录会降低资料质量。
4. **大批生成图片与中间文件**
   - 对阅读者未必友好，也会显著放大仓库体积。

## 适合怎么用

- 查 `港口布局图/` 的图时，对照港口序号、区域与经纬度
- 做世界地图、航线、区域分布相关的二次整理
- 想研究剧情结构、六位主角之间的交叉关系时，先看本目录摘录，再决定是否深入原仓库
- 做 mod / wiki / 数据库时，把这里当作导读索引，而不是唯一来源
- 如果后面真要做复刻，先看 [`复刻可用数据清单.md`](复刻可用数据清单.md)
- 如果后面真要继续自己出数据，接着看 [`逆向算法与提取脚本总览.md`](逆向算法与提取脚本总览.md)
- 如果后面真要把这些脚本改成你自己的工具链，继续看 [`脚本可移植改造清单.md`](脚本可移植改造清单.md) 和 [`最小工具包设计.md`](最小工具包设计.md)
- 如果你想直接开始迁移主链脚本，先看 [`toolkit/README.md`](toolkit/README.md)
