## 数据来源补充

本目录前半部分仍以玩家向资料整理为主；其中 [`逆向资料/`](逆向资料/) 是一个单独来源层，当前主要吸收自 [`dongzhang84/sea2_demo`](https://github.com/dongzhang84/sea2_demo) 的公开研究成果。

为避免把“原始导入”和“本仓库二次整理”混在一起，`逆向资料/` 内部又拆成：

- `imported/sea2_demo/`：原始导入文件
- `generated/`：本仓库二次提炼出的 CSV / JSON
- 说明类 Markdown：导读、复刻清单、系统蓝图

具体文件来源与置信度见 [`逆向资料/manifest.json`](逆向资料/manifest.json)。
