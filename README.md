## 平台与运行

- **二代 MD**：[OpenEmu](https://openemu.org/) / 任一 Mega Drive 模拟器
- **二代 PC**：[DOSBox-X](https://dosbox-x.com/)（macOS 上原版 DOSBox 会卡在密码输入后）
- **三代**：Windows-only，需 DOSBox + Win95 / Wine

二代修改器仅支持 macOS，见 [`二代/修改器/README.md`](二代/修改器/README.md)。

## 二代版本怎么选

中文能找到的二代有 MD（Sega Mega Drive）和 PC（DOS）两个主流平台，PS 和 SS 版与 MD 大同小异。MD 与 PC 的取舍：

- **PC 优势**：分辨率高、港口画面风格更丰富、头像 / 物品 / 发现物图标更写实；
- **MD 优势**：剧情演出有动画、模拟器存档 / 快进方便、海战敌我显示更清楚、单挑机制有意思（卡片选择 vs PC 无脑力劈）、港口一键自动补给。
- 综上 **初次游玩推荐 MD 版本**，汉化推荐 **意志之路** 系列，目前各版本中最佳的。
- SS 版画质最佳但模拟器跑得慢、贴图偏脏，不推荐。

[意志加强版](https://www.kxb4u.com/yzzl/forumdisplay.php?fid=9) 改动很多，部分合理但魔改也不少，不再是原版风格。喜欢挑战 / 新内容的可以玩，见 [`二代/强化版/`](二代/强化版/)。
