# sea2_demo 图片素材导入

这里保存的是从 `sea2_demo` 中挑出的、对《大航海时代 II》复刻最有价值的**原始提取图片素材**。

## 为什么这一批也重要

- 上一轮导入的是结构化 JSON 和导读，解决的是“数据层”。
- 这一轮导入的是原始图片素材，解决的是“资源层”和“视觉校对层”。
- 其中很多内容和本仓库已有图并不重复：例如 `ports_original/` 是从游戏资源直接解出的港口图，而 `二代/港口布局图/` 是已有玩家整理并加了说明文字的港口图。

| `ships/` | 32 | 船只精灵 |

| `world/` | 1 | 世界地图图像 |
| `charts/` | 7 | 剧情 / 关系 / 结构图 |

## 有意不导入的内容

- `game/assets/events/` 里的 8 张事件图标：更偏 demo UI 占位，不是原始游戏资源核心素材。
- `icon.svg`：项目图标，不属于《大航海时代 II》原始素材。
- Godot `.import` 元数据文件：对本仓库资料用途不大。
