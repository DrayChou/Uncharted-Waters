## 简介

二代所有贸易路线按「单趟收益」从高到低排序的参考表（数据为物价 100% 且不折算的满状态收益）。前几名都是黄金 - 白银长途倒卖路线，是二代经典暴利玩法。

> **注意**：表中收益为满状态参考，实际游戏中会受物价波动、季节、港口工业值、海战损失等影响下降。

### 实用速查

**黄金线（最暴利）**：
- 委拉克路斯 / 里约热内卢黄金 → 卢卑克白银（收益 540+，长航程）
- 委拉克路斯 / 里约热内卢黄金 → 长崎白银（收益 500+，超长航程）
- 委拉克路斯 / 里约热内卢黄金 → 热那亚白银（收益 416，长航程）
- 东非黄金 → 卢卑克 / 长崎 / 热那亚白银（收益 282-406）
- 西非黄金 → 卢卑克白银（收益 236，中短航程）

**丝织品线（欧亚贸易）**：
- 长安丝织品 → 安特卫普毛织品（250）
- 长安丝织品 → 那不勒斯毛织品（171）
- 长安丝织品 → 伊比利 / 麻布（157-209）
- 泉州丝绸 → 伦敦羊毛（179）
- 北欧棉织品 → 长安丝织品（210）

**新手友好线**（短航程 + 低风险）：
- 18. 亚丁琥珀 → 刻里马纳玳瑁甲（226，**短**）
- 36. 开罗美术品 → 麦加麝香（152，**超短**）
- 69. 雅典美术品 → 伊斯兰绒毯（76，超短）
- 41. 尼古西亚铜矿石 → 突尼斯铁矿石（132，短）

**意大利主角开局线**（无法进入伊斯兰港）：
- 94. 马德拉韦砂糖 → 伊比利橄榄油（58，超短）
- 配合 36 号线的开罗 → 麦加（伊斯兰区也受限制，但已不在伊比利亚屏障内）
