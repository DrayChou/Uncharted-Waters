# 二代 意志加强版

> 大航海时代 2 的著名改版「意志之路」出品，覆盖 MD（Sega Mega Drive）和 PC 两大平台。

## 简介

意志加强版系列以原版为基础，做了大量魔改：从 E 系列的核心重置（新增大明、外传主角）到 F 系列的系统扩展（捕鱼/道观/分舰队/海怪战/赌场/通婚系统/女鬼/挑战赛/船首像/船强化等）。本仓库收录目前能找到的最新版本：

| 平台 | 版本 | 状态 |
|---|---|---|
| MD | 1.9E6 | 2021.6 发布，E 系列终版，**新增大明、外传主角** |
| MD | 1.9F5+ 修正 2 | 最新，2024.4.8 发布 |
| PC | 1.9CⅢ 修正版 | 2008 年最后发布 |

> 主仓库 `README.md` 对加强版的评价是「不太推荐，魔改多不是原版风格了」，但作为一款历经 18 年仍在更新的改版，仍有独特的玩法和收藏价值。

## 文件说明

### 文档

- **[1.9E6论坛整理.md](1.9E6论坛整理.md)**：把 `1.9E6` 论坛帖里真正有增量价值的内容集中起来，主要包括：前中期玩法经验、顺风相送/指南正法的论坛实测口径、大明剧情完成度、已知 bug 线索，以及为什么旧 EC 码不能直接用于 `1.9E6`。
- **[1.9E6修改器整理.md](1.9E6修改器整理.md)**：整理外部 `1.9E6` Windows 修改器说明书里的关键规则，重点是哪些改法安全、哪些会触发反作弊、以及对我们继续做 `1.9E6` 金手指/状态修改的启发。
- **[1.9E6交易商品整理.md](1.9E6交易商品整理.md)**：把 `全部交易商品.xlsx` 提炼成仓库内可复用资料，包含表结构说明、价差最高的品类、低本金起步路线线索，以及结构化数据入口。
- **[F5+系统详解.md](F5+系统详解.md)**：F5+ 版本所有系统的详细说明（早期 txt 转 MD），包含 37 种船规格表、10 种海上事件、极端气候、买卖公式、海战 AI、刷分/刷钱技巧、亲王国策等。
- **[F5+系统详解-论坛原帖.md](F5+系统详解-论坛原帖.md) + .pdf**：Ken_Tse 在 2024.3.24 论坛发布的最新版本系统详解，**与 MD 版内容高度重合但更新**。建议直接看 PDF。
- **[更新历史.md](更新历史.md)**：从最早的 1.9E 到 1.9F5+ 修正 2 的所有版本更新日志，按版本倒序排列，方便追溯每个版本的变更。
- **[数据/1.9E6全部交易商品.csv](数据/1.9E6全部交易商品.csv) / [数据/1.9E6全部交易商品.json](数据/1.9E6全部交易商品.json)**：从 `1.9E6` 工具包的 Excel 直接提取出的原始结构化数据，适合后续做路线计算、价格验证和脚本分析。
- **[tools/patch_1_9e6_state_supply.py](tools/patch_1_9e6_state_supply.py)**：直接读取 `1.9E6` 的即时存档，检查或补丁式改写第一条船的 `粮食/饮水`，适合当前这种“先把存档救活再继续玩”的场景。

### ROM

ROM 文件按 `平台-版本.扩展名` 命名规则。仓库**只保留 7z 压缩包**（已对 1.9E6 的 RAR 重压缩为 7z），解压后即可得到原始 `.BIN`：

| 文件 | 平台 / 模拟器 | 压缩格式 | 解压大小 |
|---|---|---|---:|
| `MD-1.9E6.7z` | MD / Sega Mega Drive 模拟器 | 7z (LZMA2) | 3.5 MB |
| `MD-1.9F5+修正2.7z` | MD / Sega Mega Drive 模拟器 | 7z (LZMA2) | 4.2 MB |
| `PC-1.9CIII修正版.7z` | PC / DOSBox-X | 7z (LZMA2) | 2.0 MB |

> 三个版本合计约 4.9 MB，平均压缩比约 2.8×。7z 比 RAR 多 5-10% 的压缩率，且工具更通用（macOS / Windows / Linux 全部原生支持）。
>
> macOS 解压命令（任选其一）：
> - `unar <文件名>`（系统已装）
> - `7z x <文件名>`（brew install p7zip）
> - The Unarchiver / Keka 图形界面双击

## 运行

- **MD 版**：用 [OpenEmu](https://openemu.org/)、[Genesis Plus GX](https://bitbucket.org/eke/genesis-plus-gx)、[Fusion](https://www.emulator-zone.com/doc.php/genesis/fusion.php)、全能模拟器（Picodrive 核心）、MD.EMU、小鸡模拟器等任一 Mega Drive 模拟器。MAC 用户首推 OpenEmu。
- **PC 版**：用 [DOSBox-X](https://dosbox-x.com/) 运行（在原版 DOSBox 上会卡在输入密码后的画面）。

> 详细模拟器选择见 1.9F5+ 修正版更新说明中的兼容性列表。
