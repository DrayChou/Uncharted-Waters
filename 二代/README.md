# 二代 大航海时代

> 大航海时代 II（光荣，1993）。本仓库的资料主体，包括地图、港口布局图、船只图标、人物 / 港口 / 船只数据、海战机制图、强化版（意志加强版）资源、以及 macOS 上的存档修改器。

## 平台与版本

中文能找到的二代有两个主流平台：

| 平台 | 优点 | 缺点 | 推荐场景 |
|---|---|---|---|
| **MD（Sega Mega Drive）** | 剧情演出有动画、模拟器存档 / 快进方便、海战界面友好、操作可全键盘 | 港口贴图小、列表查找略繁琐 | 初次游玩推荐，模拟器 [OpenEmu](https://openemu.org/) |
| **PC（DOS）** | 分辨率高、港口风格更丰富、头像更写实 | 海战界面略弱、macOS 上需 [DOSBox-X](https://dosbox-x.com/) | 喜欢精致港口画面 |

PS / SS 版与 MD 大同小异。SS 画质最佳但模拟器跑得慢、贴图也偏脏。

汉化版本推荐 **意志之路** 系列：
- 原版纯汉化 → 适合还原原作体验
- 意志加强版 → 在原版上做了大量魔改（新增主角、捕鱼、道教、分舰队、海怪、赌场等），见 [`强化版/`](强化版/)
